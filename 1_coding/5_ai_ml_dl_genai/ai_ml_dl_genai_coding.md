# AI/ML Coding Practice Platforms

*   [**Deep-ML**](https://www.deep-ml.com/problems): Excellent resource for linear algebra, calculus, and ML basics in Python.
*   [**LeetCode (ML Section)**](https://leetcode.com/problemset/machine-learning/): Newer section with Pandas and ML specific questions.
*   [**Kaggle Learn**](https://www.kaggle.com/learn): Interactive tutorials for Pandas, Feature Engineering, and Deep Learning.
*   [**PyTorch Recipes**](https://pytorch.org/tutorials/recipes/recipes_index.html): Bite-sized PyTorch code examples.
