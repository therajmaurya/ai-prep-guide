# SQL for Data Science

## Core Ideas
- SELECT, FROM, WHERE, GROUP BY, HAVING.
- JOINs (Inner, Left, Right, Full).
- Window Functions (RANK, LEAD, LAG).
- CTEs (Common Table Expressions).

## Resources
- [PostgreSQL Documentation](https://www.postgresql.org/docs/)
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/)

## Practice
- Write a query using a Window Function to calculate running totals.
- Optimize a query using indexes.

## Questions
### Mid Level
- Difference between `WHERE` and `HAVING`.
- Explain `UNION` vs `UNION ALL`.

### Senior Level
- Discuss database normalization vs denormalization for analytical workloads.
